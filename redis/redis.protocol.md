# RESP

一个字符串的协议，标记了类型，数据长度信息。

1. strings
2. integers
3. arrays
4. errors

数据类型依赖首个字节。

1. 单行字符串，首字节`+`
2. 多行字符串，首字节`$`
3. integers，首字节`:`
4. errors，首字节`-`
5. arrays，首字节`*`

总是以`\r\n`结束。

## 单行字符串

1. `+`号后面跟一个非换行的字符串，以`\r\n`结束。

**如下示例**

```
+OK\r\n
```

实际内容应该去掉`+`和`\r\n`。

## 一个错误类型

1. `-`号后面就是错误本身的信息（其实就是一个字符串）和普通的[单行字符串](#单行字符串)是一样的，
区别就是，实现的<ins>客户端</ins>会把这样的数据当成错误或异常处理

**如下示例**

```
-这里是一个错误信息\r\n
```
